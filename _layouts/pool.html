About This App

Hello! I'm Meagan Truglio the creator of the practice app. I'm a web developer, systems administrator and pool player based in San Diego. I built this app because wanted an app to track my practice sessions with fellow community members and measure performance over time. Three key features I wanted to include were multiple game modes, all user data stored on user device, and the ability to score multiple games in airplane mode or without internet.
