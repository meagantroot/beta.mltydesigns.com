About This App

The Practice App is a progressive web app "PWA" for scoring 8, 9 and 10 ball pool. To preserve privacy and improve speed, scoring data is stored locally. This means your data never leaves your device. The Practice App can be installed to your device to run just like a native app giving optimal user experience.

From the Creator

Hello! I'm Meagan Truglio the creator of the practice app. I'm a web developer, systems administrator and pool player based in San Diego. I built this app because wanted to track my practice sessions with fellow community members and measure performance over time. Three key features I wanted to include were multiple game modes, all user data stored on user device, and the ability to score multiple games in airplane mode or without internet.
